# 📊 Customer Churn Analysis Dashboard | Power BI

## 🚀 Project Overview
Developed an interactive **Customer Churn Analysis Dashboard** in **Power BI** to analyze customer retention patterns and identify the key factors influencing churn. The dashboard transforms raw customer data into actionable insights, enabling businesses to make informed decisions for improving customer satisfaction and reducing customer attrition.

## 🎯 Objectives
✔️ Analyze customer churn trends and behavior  
✔️ Identify high-risk customer segments  
✔️ Understand the impact of demographics, contracts, and services on churn  
✔️ Support data-driven customer retention strategies  

## 🛠️ Tools & Technologies
🔹 Power BI  
🔹 CSV Dataset  
🔹 Power Query (Data Cleaning & Transformation)  
🔹 DAX (Measures & KPIs)  

## 📌 Key Dashboard Features
✨ Interactive and user-friendly visualizations  
✨ Customer segmentation analysis  
✨ Churn rate tracking and KPI monitoring  
✨ Contract type and payment method insights  
✨ Service subscription analysis  
✨ Dynamic filtering and drill-down capabilities  

## 🔍 Insights Generated
📈 Identified customer groups with the highest churn probability.  
📈 Analyzed the relationship between customer tenure and churn.  
📈 Evaluated the impact of contract types and payment methods on customer retention.  
📈 Uncovered trends that help businesses improve customer loyalty and reduce revenue loss.  

## 📂 Dataset Information
The dataset includes customer demographics, account details, subscribed services, contract information, tenure, payment methods, and churn status.

## 🎓 Project Outcome
This dashboard provides a comprehensive view of customer behavior and churn patterns, helping organizations proactively address retention challenges and enhance business performance through data-driven decision-making.
